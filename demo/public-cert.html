Client Certs - Public Example

This is an implementation of the system at https://github.com/epw/https-client-cert-manager. This page is provided to try out the system. It will let you generate a real client certificate for this Web server. Since it is just a testing server, all you will really be able to do with this certificate is generate more certificates, but that should let you explore what it is like to use them across different devices.

Normally, you would probably want some other method of vetting users and knowing who they are, so that you can assign them meaningful names. As it is, this will simply generate a random 10-letter string as a name. This even means that there is a tiny (26-10) chance that your name will be the same as someone else's, and so, even though you will have different public keys, this server's standard identification software will confuse you. Again, a real server implementing this system should be smarter.
